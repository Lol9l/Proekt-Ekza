## 6. Установка и настройка

### Требования:

- Windows 10/11
- .NET Framework 4.7.2+
- Visual Studio 2019+

### Шаги установки:

1. Распаковать архив проекта
2. Открыть `Travel-Company.WPF.sln` в Visual Studio
3. Собрать проект и запустить
4. При необходимости отредактировать строку подключения в `Settings.cs`
